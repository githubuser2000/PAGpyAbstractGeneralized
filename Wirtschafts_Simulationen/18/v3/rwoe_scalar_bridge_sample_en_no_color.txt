===================================================================================================
Simulation part 15: Scalar bridge: relation table becomes one market number
===================================================================================================
╭─────────────────────────────────────────────────────────────────────────────────────────────────╮
│                                        What is simulated                                        │
├─────────────────────────────────────────────────────────────────────────────────────────────────┤
│  The relation table is converted into a single market-equivalent number by multiplying every    │
│  cell with a row factor and a column factor, then summing all contributions.                    │
╰─────────────────────────────────────────────────────────────────────────────────────────────────╯
╭─────────────────────────────────────────────────────────────────────────────────────────────────╮
│                                      Why this is simulated                                      │
├─────────────────────────────────────────────────────────────────────────────────────────────────┤
│  This matters because it shows exactly how the stronger relational system can be translated     │
│  back into a classical market economy without pretending that the original table was already    │
│  just a number.                                                                                 │
╰─────────────────────────────────────────────────────────────────────────────────────────────────╯

│      │ number.                                                                                  │
