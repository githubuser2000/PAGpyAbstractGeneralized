Simulationsteil 01: Anatomie der relationalen Währung
====================================================================================================
╭──────────────────────────────────────────────────────────────────────────────────────────────────╮
│                                        Was simuliert wird                                        │
├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│  Dieser Teil baut ein kleines RWB mit Relationszeilen, Anspruchszeilen und Gültigkeitsschicht.   │
│  Ziel ist zu zeigen, dass das Währungsobjekt ein tabellenartiges Bündel ist und keine Zahl.      │
╰──────────────────────────────────────────────────────────────────────────────────────────────────╯
╭──────────────────────────────────────────────────────────────────────────────────────────────────╮
│                                     Warum das simuliert wird                                     │
├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│  Das wird zuerst simuliert, weil alle späteren Märkte ein konkretes Währungsobjekt brauchen.     │
│  Zeilenzählen allein wäre zu schwach; das Bündel braucht Rechts-, Zeit- und Risikostruktur.      │
╰──────────────────────────────────────────────────────────────────────────────────────────────────╯
╭──────────────────────────────────────────────────────────────────────────────────────────────────╮
│                                   Wie dieser Teil zu lesen ist                                   │
├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│  Hier werden wirklich nur die Kürzel erklärt, die unten in diesem Teil vorkommen. Einheiten      │
│  sind sichtbare Markierungen; sie sind nicht selbst die Währung.                                 │
╰──────────────────────────────────────────────────────────────────────────────────────────────────╯
Kürzel, die nur in diesem Teil vorkommen
┌────────┬────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐

│ RWÖ    │ Relationale Währungsökonomie: das Gesamtsystem, in dem Währung ein strukturiertes Relationenbündel statt einer einzelnen Zahl ist. │
│ RWB    │ Relationales Wertbündel: eine handelbare Tabelle aus Ansprüchen, Pflichten, Risiken, Zeitfenstern und Nachweisen.                  │
│ RID    │ Relationskennung: die nachvollziehbare Bezeichnung einer einzelnen Zeile im Bündel.                                                │
│ VAL    │ Gültigkeitsschicht: der Teil, der sagt, ob eine Relation rechtmäßig, aktuell und durchsetzbar ist.                                 │

Simulationsteil 02: Markt-Matching ohne einzelnen Preis
====================================================================================================
╭──────────────────────────────────────────────────────────────────────────────────────────────────╮
│                                        Was simuliert wird                                        │
├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│  Dieser Teil simuliert, wie Annahmetabelle und Gebotstabelle nach Passung verbunden werden,      │
│  nicht nach einem einzigen Preisschild.                                                          │

│  Eine relationale Währung wird nur nützlich, wenn Märkte sie verrechnen können. Matching zeigt,  │

Simulationsteil 03: Produktionskette als Relationsfluss
====================================================================================================
╭──────────────────────────────────────────────────────────────────────────────────────────────────╮
│                                        Was simuliert wird                                        │
├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│  Dieser Teil verfolgt Eingänge, Kapazität, Zwischenarbeit und Durchsatz durch eine               │
│  Produktionskette, in der jeder Schritt Rechte und Pflichten trägt.                              │

│  In der Produktion wird das System wirtschaftlich stärker: Es sieht Engpässe, Pflichten und      │
│  ungenutzte Kapazitäten früher als ein bloßes Preissignal.                                       │

├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│  Der Schluss erklärt, warum das System stärker ist als eine bloße Marktwirtschaft und warum      │

╰──────────────────────────────────────────────────────────────────────────────────────────────────╯
╭──────────────────────────────────────────────────────────────────────────────────────────────────╮

├──────────────────────────────────────────────────────────────────────────────────────────────────┤
│  Hier werden wirklich nur die Kürzel erklärt, die unten in diesem Teil vorkommen. Einheiten      │
