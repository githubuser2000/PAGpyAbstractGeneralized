# BRICS諸国がスイスのように直接民主制で統治されたら

そうだ。ただし、それは**単純に「民主主義が増える」ことにはならない**。BRICSを**分裂させる**か、まったく別のものに変えるだろう。つまり、**直接民主制のメガ連邦／連合体**である。

最初の厳しい点はこれだ。現在のBRICSは国家ではなく、政治・外交上の調整フォーラムである。BRICSの公式説明は11の加盟国を挙げ、この枠組みを協力のフォーラムとして説明している。意思決定は市民の意思ではなく、国家間の合意によって進む。出典：[BRICS — About the BRICS](https://brics.br/en/about-the-brics)。一方、スイスでは市民が国民投票で議会決定を修正し、国民発案で憲法改正を提案できる。スイス連邦の資料は、発案と国民投票を直接民主制の二つの中心的手段として説明している。出典：[About Switzerland — Direct democracy](https://aboutswitzerland.eda.admin.ch/en/direct-democracy)。

したがって、変化はこうではない。

```text
BRICS + 投票
```

むしろこうである。

```text
BRICS諸国が特別な外交的地位を失う
→ 市民が主権者になる
→ 国家がカントン / 共和国 / デモイになる
→ BRICSが超連邦国家になる
```

## 1. まずBRICSの中核を破壊する

現在のBRICSは、非常に異なる体制同士が交渉できるから成り立っている。

```text
中国        = 中央集権的な党国家
ロシア      = 個人化された権力の垂直構造
イラン      = 神権共和制的な安全保障秩序
インド      = 中央集権傾向を持つ大衆民主主義
ブラジル    = 大統領制民主主義
南アフリカ  = 議会・大統領制民主主義
湾岸諸国    = 君主制・テクノクラート型レンティア国家
```

スイス型の直接民主制は、まさにこの統治論理を攻撃する。

直接民主制には少なくとも次が必要である。

```text
自由な討論
自由な野党
自由なメディア
自由な署名収集
公正な投票管理
司法的統制
国家による威嚇からの保護
```

それがなければ、スイス型直接民主制ではなく、単なる**人民投票的な拍手喝采**になる。

```text
体制が質問を設定する
体制がメディアを支配する
体制が票を数える
体制が結果を「人民の意思」と呼ぶ
```

それは7型民主主義ではない。**投票の外皮をまとった5型中央国家**である。

## 2. 本当にスイス型にすれば、BRICSは最強の体制に対して反転する

中国、ロシア、イラン、サウジアラビア、UAEで本物の国民発案と国民投票が行われれば、それは体制変革的である。市民は税やインフラだけでなく、権力の基盤についても間接的に投票できるようになる。

```text
検閲
一党独占
戦争政策
情報機関の権力
腐敗
補助金
軍事支出
宗教法
資本移動
労働移民
環境破壊
```

それはこういう意味である。

> 本物のBRICS直接民主制は、BRICSの改革ではない。  
> 複数のBRICS諸国内で同時に起きる民主革命である。

したがって、現実的な最初の結果はこうなる。

```text
権威主義的な加盟国はそれを阻止し、
離脱し、
手続きを操作し、
または統制された国民投票へ変えようとする。
```

## 3. カントン多数がなければ、中国とインドがすべてを支配する

スイスは国民多数だけに依存していない。憲法改正にはカントン多数も必要であり、超国家的共同体への加盟にも二重多数が必要である。出典：[ch.ch — Popular majority and majority of the cantons](https://www.ch.ch/en/votes-and-elections/votes/popular-majority-and-majority-of-the-cantons/)。

これは決定的である。

これをBRICSに移すとどうなるか。世界銀行の2024年データによれば、インドと中国の人口はそれぞれ約14.5億人、14.1億人である。出典：[World Bank Data — China and India](https://data.worldbank.org/?locations=CN-IN)。国家多数がなければ、インドと中国は共同でBRICS全体の国民投票をほぼ支配できる。

その場合、制度は直接民主的・連邦的ではなく、人口的に帝国的になる。

```text
インド + 中国
      ↓
その他のBRICS
```

これは政治的に爆発的である。ブラジル、ロシア、南アフリカ、イラン、エジプト、エチオピア、インドネシア、サウジアラビア、UAEは、自国の主権が印中の人口多数によって覆されることを決して受け入れないだろう。

したがって、BRICS版スイスには少なくとも次が必要である。

```text
国民多数
+
国家多数
+
マクロ地域多数
+
基本権審査
```

こうではない。

```text
40億票 → 単純多数
```

こうである。

```text
市民多数
+ 加盟国多数
+ 世界地域 / デモイ多数
+ 憲法裁判所
```

そのとき初めて7になる。

## 4. BRICSを国家ブロックからデモイクラシーへ変える

正しい用語は単純な意味での「民主主義」ではなく、**デモイクラシー**である。

```text
一つの人民ではなく、
多数の人民が、
一つの人民へ融合せずに、
共に統治する。
```

それは次に近い。

```text
スイス
+ 欧州連合
+ インド
+ 直接民主制
+ 強い憲法裁判所
+ 多言語の権力分担
```

制度構造はこうでなければならない。

```text
市民院：
人口比例

国家院：
各BRICS国が強い重みを持つ

デモイ院：
言語、少数派、先住民集団、宗教空間

直接民主制：
発案と国民投票。ただし署名要件は分散型

憲法裁判所：
基本権、少数派保護、権限問題を審査

7人または11人の執政委員会：
単独指導者なし、超大統領なし
```

そうでなければ、直接民主制はただちに最大勢力の機械になる。

## 5. 肯定的な結果は巨大である

もし本当に成功すれば、歴史的に新しいものが生まれる。

### 第一：BRICS政治の脱軍事化

政府は地政学的象徴を好む。

```text
反西側レトリック
首脳外交
通貨プロジェクト
軍事協力
戦略軸
```

市民はしばしばより冷静に投票する。

```text
生活費
雇用
腐敗
医療
住宅
教育
安全
インフラ
環境
```

BRICSは反西側の権力ブロックとしてよりも、**社会経済的な開発連合**として機能するようになる。

### 第二：腐敗への下からの圧力

直接民主制はエリート・カルテルを乱すことができる。自動的にではないが、構造的にである。

市民が大型プロジェクト、条約、補助金、特権に対して国民投票を発動できれば、次に圧力がかかる。

```text
オリガルヒの特権
監視なき国有企業
資源レント
地域同意なきメガプロジェクト
秘密の対外条約
```

### 第三：新しい世界的正統性

直接民主制のBRICSはもはや単にこれではない。

```text
「グローバル・サウス対西側」
```

むしろこうである。

```text
「数十億の市民が自ら投票する」
```

これは巨大な象徴的力を持つ。BRICSが規模だけでなく正統性でも議論できるようになるため、西側民主主義諸国に圧力をかける。

## 6. 否定的な結果も過酷である

### 第一：恒久的な閉塞

スイスの直接民主制が機能する理由の一つは、国家が比較的小さく、豊かで、制度的に安定し、連邦制に習熟していることにある。スイスでは政策案件について年におよそ4回投票が行われ、近年数十年の平均投票率は40%を少し上回る。出典：[About Switzerland — Direct democracy](https://aboutswitzerland.eda.admin.ch/en/direct-democracy)。

BRICS規模では、これは次へ転化しうる。

```text
絶え間ない投票キャンペーン
恒常的な偽情報
翻訳の混乱
国民的動員
宗教的分極化
外交政策の妨害
```

### 第二：国民投票が地政学的武器になる

各大国は他の加盟国の投票に影響を与えようとする。

```text
中国はディアスポラとプラットフォーム空間に影響する
ロシアは安全保障ナラティブに影響する
インドは人口多数に影響する
湾岸諸国は資金で影響する
イランは宗教ネットワークで影響する
西側アクターは対抗キャンペーンに影響する
```

この規模では、極端な透明性規則がなければ直接民主制はすぐに操作可能になる。

### 第三：少数派の恐怖

単純な直接民主制は少数派にとって危険になりうる。

だからスイスでは憲法問題に国民多数だけでなくカントン多数も必要である。BRICSではさらに進む必要がある。

```text
単純多数で基本権を廃止できない
多数で言語を排除できない
多数で宗教の権利を奪えない
多数で地域を収奪できない
単純多数で国家自治を廃止できない
```

それがなければ、直接民主制は多数派の武器になる。

## 7. 角度モデルとして

現在のBRICSはおおよそこうである。

```text
国家 ↔ 国家 ↔ 国家
       政府の合意
```

つまり**6型コンフェデレーション**に近いが、内部には多くの権威主義的空間がある。

スイス式の直接民主制BRICS連合は、これをこう変える。

```text
市民 ↔ 国家 ↔ 地域 ↔ デモス ↔ 裁判所 ↔ 未来 ↔ 執行部
```

これは**7型民主主義**である。

しかし移行は非常に危険である。

```text
悪く作れば：

人口多数
        ↓
中国/インド支配
        ↓
小国が反発
        ↓
分裂または中央集権化
```

```text
正しく作れば：

国民多数
+ 国家多数
+ デモイ保護
+ 基本権
+ 地域自治
+ 直接的修正
        ↓
大陸的ロゼット民主主義
```

## 8. 最も可能性の高い結果

もし明日導入すれば：

```text
1. 権威主義的加盟国は真に自由な投票を拒む
2. 民主的加盟国は正統性をめぐって争う
3. 中国とインドは人口的支配極として恐れられる
4. 小規模加盟国は拒否権を要求する
5. システムは閉塞する
6. BRICSは崩壊するか、偽の直接民主制になる
```

短期的には：

> **BRICSを民主化するより、むしろ引き裂く。**

しかし法の支配、連邦制、複数多数決、本物の報道自由とともに、数十年かけて清潔に構築すれば、非常に強力なものに至る。

> **グローバル・サウスの直接民主制メガ連合体。**

それは大きなスイスではない。むしろこうである。

```text
スイス = アルプス規模の直接民主制

BRICSスイス = 惑星的な前国家としての直接民主制
```

## 最終公式

```text
法の支配なし：
BRICS直接民主制 = 人民投票的ファサード

国家多数なし：
BRICS直接民主制 = インド/中国支配

少数派保護なし：
BRICS直接民主制 = 多数派の専制

自由メディアなし：
BRICS直接民主制 = プロパガンダ機械

すべてが揃えば：
BRICS直接民主制 = メガ連合体としての7型民主主義
```

最も正確な一文：

> **BRICSがスイスのように直接民主制で統治されるなら、BRICSはBRICSであることをやめるか——権威主義であり続けることをやめる。**
